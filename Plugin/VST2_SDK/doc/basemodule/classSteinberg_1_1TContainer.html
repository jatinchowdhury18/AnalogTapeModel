Detailed Description

template<class T>
class Steinberg::TContainer< T >

Template container base class (abstract).

This class defines the basic interface for any container implementation in this module.
Please note that:

  • total / size : You should always use TContainer::total () to get the number of items in the container and NOT TContainer::size ().
  • TContainer::error () : There is a special item that is returned in error cases when a method returns a T&. This item is created using the T default constructor. So if you use a container template with build in types like 'int' or pointers, you might need to initialize the error object yourself with a valid 'empty' value. This cannot be done automatically, since not every class has a constructor taking a NULL as argument! For example: 
    TArray<int> myArray;
myArray.error () = -1;
See also:
TIterator, TArray, TLinkedList, TDLinkedList
